Список планируемых функций бота (обновляемый):
1) Приветствие пользователя - Выполнено
2) Реакция на действие пользователя (команду и просто сообщение) - Выполнено
3) Чтение информации из файла и отправка пользователю - Выполнено
4) Взаимодействие с API - выполнена для простейших ботов 
Для бота "Угадай перевод" взаимодействие с API не реализовано, т.к. 
API русско-английских словарей платные, в учебных целях нет смысла их оплачивать, 
вместо этого происходит чтение слов из файла - Dictionary

5) Добавление многопользовательности
6) Проведение опроса пользователя (с целью выявить интересы)
7) Анализ результатов опроса
8) Предложение подходящих (на основании анализа) туров
9) Запись данных пользователя в БД (в первом приближении достаточно использовать таблицу Excel)
10) Перенос бота на базу удаленного сервера
