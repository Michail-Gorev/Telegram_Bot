Учебный проект телеграмм-бота. 
Планируемый результат работы - бот, предлагающий подходящую конкретному человеку 
подборку туров в Москву, с подробной информацией о гостиницах, свободном времени и т.д.

По ходу реализации проекта пишется большое количество примитивных ботов с целью отработки и 
развития понимания структур языка PYTHON (в частности - библиотеки aiogram), API и прочих инструментов, необходимых для
создания финального продукта. 
Фиксация хода работы осуществляется с помощью GIT-а, а также в файле Check-List
